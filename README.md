
# Gumiverzum v.3
Készítette: Krammer János és Pálfy Alexander
A projekt egy gumiszerviz online felületére terjed ki
Az oldalon egy működő időpontfoglaló, és egy tevezett webshop található ami a későbbi fejlesztésnek az első állomása lesz.
További információ a dokumentációban található ahol sokkal részletesebb leírást adunk a projektről.
